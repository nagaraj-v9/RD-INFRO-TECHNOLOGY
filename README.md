Project Objective

The objective of this project is to analyze airline passenger satisfaction data by performing Data Cleaning, Exploratory Data Analysis (EDA), and Feature Engineering. The project identifies important factors influencing passenger satisfaction and prepares the dataset for future machine learning applications.


Dataset

- Airline Passenger Satisfaction Dataset
- More than 100,000 passenger records
- 24 features after data cleaning
